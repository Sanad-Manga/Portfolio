# Ahmed Sanad — Personal Portfolio

A personal portfolio website built from scratch and deployed on Vercel. Showcases my projects, skills, and education as a Computer Science student at GIU.

🔗 **Live site:** [ahmedsanad.vercel.app](https://ahmedsanad.vercel.app)

---

## Features

- Responsive layout — works on mobile, tablet, and desktop
- Dark / light mode toggle with localStorage persistence
- Mobile navigation menu
- Project cards with status badges (Live, In Progress, In Development)
- Smooth scroll and back-to-top button
- Vercel Analytics integration

---

## Tech Stack

- **HTML5 / CSS3 / Vanilla JavaScript** — no frameworks, built from scratch
- **Font Awesome** — icons
- **Google Fonts** — typography
- **Vercel** — hosting and analytics

---

## Project Structure

```
Portfolio/
├── index.html        # Main page
├── styles.css        # All styling
├── script.js         # Dark mode, nav, scroll behavior
├── menu.svg          # Mobile menu icon
├── close.svg         # Mobile menu close icon
└── package.json      # Vercel config
```

---

## Running Locally

No build step needed — just open index.html in your browser, or use Live Server in VS Code.

```bash
npx serve .
```

---

## Deployment

Deployed automatically via Vercel on every push to main.
